Frequently asked questions

What tools/programs will be needed for the workshops?

You will need a computer or laptop with internet access to participate in the program. We recommend using Replit, an online coding platform that provides an integrated development environment (IDE) for Python and other programming languages. Replit is user-friendly, can be accessed from any device including a web browser, and allows for collaborative coding and project sharing. Additional instructions on Replit setup and account creation will be provided prior to the meeting.

How will the workshop be held?

The workshop will be conducted online using Zoom. We will provide you with the necessary details and instructions on how to join the Zoom session prior to the workshop through email and to our Discord Server. The instructor will share their screen to demonstrate coding concepts, guide students through exercises, and provide answers to any questions. Join the Discord Server using this link.
